# django-chat
A twosome chat with lobby on Django

## System requirements

Ubuntu 14.04

python 2.7.6

Django 1.9.7

tornado 4.4

## Run

1. Run django server: python manage.py runserver
2. Run tornado server: python chat/app.py

## Environment variable

DJANGO_PORT (default=8000)

DJANGO_HOST (default="0.0.0.0")
